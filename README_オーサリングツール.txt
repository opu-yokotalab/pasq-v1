オーサリングツール setpano.htmについて

※近傍情報付のPCDファイルの読込や、近傍パノラマ・切替範囲の表示はできますが、
　手動での近傍情報の設定は諸問題により、行えないようにしてあります。
　このツールで配置を行う場合は、近傍情報付与前のPCDファイルを使用してください。

プロトタイプに機能追加を繰り返したため、非効率・解読困難(本人も)になっており、
今後必要なら改良よりは新しく作り直したほうがいいと思います。


○簡単な操作説明
・読込、保存
　XMLファイル全てが表示されるため、PCDファイル以外は選択しないように

・方位情報設定
　撮影方位を統一した場合、ここで事前に方位情報を設定しておく
　ブラウザのリロードで初期化されるので注意
　マーカークリックで表示される情報提示部分で個々の調整も可能

・スタートパノ設定
　PasQ起動時に表示されるパノラマ画像を選択
　既に設定してある場合は現在のものが青いマーカーで表示される

・パノラマ配置
　パノラマ画像をリストから選択し撮影地点に配置
　配置済みの画像はわかるようになっており選択も不可
　配置したパノラマ画像を移動させる際は、マーカーを一度クリックしてから(事故防止)

・一括配置
　直線状に等間隔で撮影したパノラマを一括で配置する際に使用
　配置する画像を3個以上選択し、撮影位置の始点と終点を指定すると等間隔で配置

・近傍パノラマ表示
　近傍情報付のPCDファイルであれば、マーカーをクリックすることで
　近傍パノラマ、切替範囲を見ることが可能
　近傍情報付PCDファイルは読込のみで使用してください
